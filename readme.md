## MovieDB API

API feita no minicurso de .NET da PUC Minas Poços de Caldas no segundo semestre de 2024.

## Funcionalidades

A API permite adicionar, remover, atualizar e buscar atores, filmes e séries.

## Requisitos

### .NET 8
### EntityFrameworkCore
### MySQL
### Swagger

Adicione a connectionString de acordo com seu banco, exemplo: 

`   "DefaultConnection": "Server=localhost; Database=moviedb; User=root;Password=;"
`
